# Account Takeover Response Steps

## Immediate Actions
- Temporarily restrict or lock the account if risk is high
- Require identity verification
- Invalidate active sessions
- Force password reset
- Review MFA settings and recent account changes

## Investigation Actions
- Review authentication history
- Review device and location data
- Check for profile or contact changes
- Review transaction or activity history
- Determine whether similar patterns affect other accounts

## Recovery Actions
- Restore legitimate account settings
- Reverse unauthorized changes where possible
- Notify the user
- Monitor the account for repeat activity

## Long-Term Controls
- Strengthen MFA protections
- Add alerts for high-risk account changes
- Increase monitoring for anomalous login sequences
- Improve friction for risky sessions without over-impacting legitimate users
